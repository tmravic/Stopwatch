# Stopwatch
This is a stopwatch program which handles input and can do the following tasks.
・Take in a time 
・Countdown
・Pause
・Reset its time

Time can be incremented in intervals of:  
・ 1 minute  
・ 10 seconds  
・ 1 second

Program can be controlled with the following buttons:

・ Start<br>
・ Stop*<br> 
・ Reset

* This pauses the the stopwatch.
