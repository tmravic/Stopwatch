# Stopwatch
This is a stopwatch program which handles input and can do the following tasks.<br> 
・Take in a time <br> 
・Countdown<br> 
・Pause<br> 
・Reset its time<br> 

Time can be incremented in intervals of:  
・ 1 minute  
・ 10 seconds  
・ 1 second

Program can be controlled with the following buttons:
・ Start<br>
・ Stop<br> 
・ Reset
